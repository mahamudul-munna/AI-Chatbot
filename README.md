# 🤖 Creamy AI Chatbot

A smooth, modern, and responsive AI chatbot built using **HTML, CSS, and JavaScript**, powered by OpenAI API.  
Designed with a creamy minimal UI and fully functional chat system with memory, typing effect, and theme toggle.

---

## ✨ Features

- 💬 Real-time AI chat responses (OpenAI API)
- 🧠 Chat memory (context-aware replies)
- ⌨️ Typing animation for AI responses
- 💾 LocalStorage chat saving (auto chat history)
- 🌙 Dark mode / Light mode toggle
- 📱 Fully responsive UI (mobile + desktop)
- 🎨 Creamy, modern UI design
- ⚡ Smooth animations & transitions
